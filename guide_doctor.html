Nhập Thông Tin Bệnh Nhân Qua Đơn Thuốc

Upload file PDF đơn thuốc – hệ thống tự xử lý TỰ ĐỘNG

  • 👆 Nhấn nút "Import Đơn Thuốc" trên Dashboard
  • 📄 Chọn file PDF đơn thuốc từ bệnh viện (1 hoặc nhiều file)
  • 🤖 Hệ thống tự đọc: Tên BN, SĐT, CCCD, chẩn đoán, danh sách thuốc
  • ✅ Tự tạo hồ sơ bệnh nhân + phác đồ + đơn thuốc ban đầu

Upload Kết Quả Đo Mật Độ Xương (MĐX)

Upload file PDF kết quả đo – hệ thống tự trích xuất TỰ ĐỘNG

  • 👆 Nhấn nút "Import Kết Quả MĐX" trên Dashboard
  • 📄 Chọn file PDF kết quả đo (1 file hoặc nhiều file)
  • 🤖 Hệ thống tự đọc: T-Score, BMD tại cổ đùi trái/phải, cột sống L1–L4
  • 📊 Tự tính FRAX (% nguy cơ gãy xương 10 năm) và lưu vào hồ sơ

Chỉnh Sửa Chẩn Đoán & Tạo Phác Đồ

Kiểm tra dữ liệu import và điều chỉnh chuyên môn

  • 👁️ Vào hồ sơ bệnh nhân → Kiểm tra chẩn đoán đã được AI đọc từ file
  • 📋 Tạo Kế Hoạch Điều Trị → Nhập mục tiêu, thời gian, dặn dò
  • 💊 Tạo Đơn Thuốc → Chọn thuốc, liều dùng, tần suất uống
  • 📅 Đặt Ngày tái khám → App tự nhắc lịch
